# SID 2026 Paper Summaries

이 저장소는 SID 2026 논문 요약본들을 정리하고 관리하기 위한 공간입니다.

## 주요 기능

- **Markdown 요약**: 각 논문의 핵심 내용을 정리한 `.md` 파일 제공.
- **웹 인터페이스**: `index.html`을 통해 브라우저에서 편리하게 논문 목록 확인 및 내용 열람 가능.
- **PPT 복사 기능**: 웹 인터페이스에서 "Copy for PPT" 버튼을 클릭하면, PPT 슬라이드에 붙여넣기 적합한 형식으로 내용이 클립보드에 복사됩니다.
- **PDF 원본**: 요약된 논문의 원본 PDF 파일 포함.

## 사용 방법

1. `index.html` 파일을 브라우저로 엽니다.
2. 좌측 사이드바에서 원하는 논문 번호를 선택합니다.
3. 내용을 확인하고, 필요 시 상단의 `Copy for PPT` 버튼을 눌러 내용을 복사합니다.
4. PPT 슬라이드에 `Ctrl+V`로 붙여넣어 활용합니다.

## 파일 구성

- `*.md`: 논문 요약 파일
- `*.pdf`: 논문 원본 파일
- `index.html`: 논문 뷰어 및 PPT 복사 도구
- `.env`: (보안 주의) GitHub 업로드용 설정 파일
